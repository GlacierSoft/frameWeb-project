jQuery EasyUI combo 组件扩展

该部分扩展由文件 jeasyui.extensions.combo.js 实现。

扩展概述

扩展基于jquery.easyui.min.js实现，主要扩展功能体现在：

  • 增加了 easyui-combo 的自定义扩展属性iconCls，表示该 combo 组件的 iconCls 图标样式类
  • 增加了 easyui-combo 的自定义扩展属性autoShowPanel，表示该 combox 组件是否在 textbox 文本显示框获取焦点时自动执行 showPanel 方法显示下拉 panel 面板
  • 增加了 easyui-combo 组件的自定义方法setIcon，用于设置 easyui-combo 控件的右侧显示图标
  • 增加了 easyui-combo 组件的自定义方法setRequired，用于设置启用或者禁用 easyui-combo 控件的表单验证功能

依赖项

  • jquery.jdirk.js
  • jeasyui.extensions.js
