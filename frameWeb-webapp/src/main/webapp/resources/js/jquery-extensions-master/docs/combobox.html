jQuery EasyUI combobox 组件扩展

该部分扩展由文件 jeasyui.extensions.combobox.js 实现。

扩展概述

扩展基于jquery.easyui.min.js实现，主要扩展功能体现在：

  • 增加了 easyui-combobox 控件默认参数 queryParams，该参数表示从远程服务器调取数据时发送的额外查询参数，该参数应该定义为一个 JSON 对象
  • 重新定义 easyui-combobox 控件的默认属性 loader，使之能处理 queryParams 参数
  • 扩展了 easyui-combobox 的自定义方法getItem，该方法用于获取当前选择了的项
  • 扩展了 easyui-combobox 的自定义方法getItems，该方法用于获取当前选择了的所有项集合

依赖项

  • jquery.jdirk.js
  • jeasyui.extensions.js

用法

扩展的属性

该扩展增加定义了如下属性：

Name Type Description Default
queryParams JSON-Object 表示从远程服务器调取数据时发送的额外查询参数 null
loader function 可以处理 queryParams 参数 null

扩展的方法

该扩展增加定义了如下方法：

Name Parameters Description Returns
load param(参数类型可以是String或者JSON Object类型) 以指定的参数查询重新加载远程数据 null
getItem 无 用于获取当前选择了的项 返回一个 JSON-Object，该 JSON-Object 为当前 easyui-combobox 数据源中的一个子项，包含 idField 和 textField 的值；
如果当前 easyui-combobox 没有选中任何值，则返回 null
getItems 无 用于获取当前选择了的所有项集合 返回一个 Array，数组中的每个元素都是一个 JSON-Object 为当前 easyui-combobox 数据源中的一个子项，包含 idField 和 textField 的值；
如果当前 easyui-combobox 没有选中任何值，则返回一个空数组。
