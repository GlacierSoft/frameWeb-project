jQuery EasyUI combogrid 组件扩展

该部分扩展由文件 jeasyui.extensions.combogrid.js 实现。

扩展概述

扩展基于jquery.easyui.min.js实现，主要扩展功能体现在：

  • 增加了 easyui-datagrid 中 editors 属性对 easyui-combogrid 的支持
  • 覆盖 easyui-combogrid 的事件 onLoadSuccess 以支持 easyui-datagrid 的自定义扩展功能
  • 覆盖 easyui-combogrid 的事件 onResizeColumn 以支持 easyui-datagrid 的自定义扩展功能

依赖项

  • jquery.jdirk.js
  • jeasyui.extensions.js
  • jeasyui.extensions.menu.js
  • jeasyui.extensions.datagrid.js
  • jeasyui.extensions.panel.js
  • jeasyui.extensions.window.js

用法

扩展的属性

该扩展增加定义了如下属性：

Name Type Description Default
editors 无 此属性用于对 easyui-combogrid 的支持 null

扩展的事件

该扩展增加定义了如下事件：

Name Parameters Description Returns
onLoadSuccess data 支持 easyui-datagrid 的自定义扩展功能 null
onResizeColumn field,width 支持 easyui-datagrid 的自定义扩展功能 null
