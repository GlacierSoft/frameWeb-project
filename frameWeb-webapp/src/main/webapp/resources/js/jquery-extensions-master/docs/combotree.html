jQuery EasyUI combotree 组件扩展

该部分扩展由文件 jeasyui.extensions.combotree.js 实现。

扩展概述

扩展基于jquery.easyui.min.js实现，主要扩展功能体现在：

  • 更改继承于 easyui-tree 的自定义属性 toggleOnClick 的默认值，使得 easyui-combotree 中 tree 组件的页节点在点击后不自动展开/折叠子节点
  • 更改继承于 easyui-tree 的自定义属性 autoBindDblClick 的默认值，使得 easyui-combotree 中 tree 组件的页节点在双击后不触发第一个右键菜单项的事件

依赖项

  • jquery.jdirk.js
  • jeasyui.extensions.js
  • jeasyui.extensions.menu.js
  • jeasyui.extensions.combo.js
  • jeasyui.extensions.tree.js

用法

扩展的属性

该扩展增加定义了如下属性：

Name Type Description Default
toggleOnClick Boolean 更改继承于 easyui-tree 的自定义属性 toggleOnClick 的默认值，使得 easyui-combotree 中 tree 组件的页节点在点击后不自动展开/折叠子节点 false
autoBindDblClick Boolean 更改继承于 easyui-tree 的自定义属性 autoBindDblClick 的默认值，使得 easyui-combotree 中 tree 组件的页节点在双击后不触发第一个右键菜单项的事件 false
