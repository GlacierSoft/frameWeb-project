jQuery EasyUI dialog 组件扩展

该部分扩展由文件 jeasyui.extensions.dialog.js 实现。

扩展概述

扩展基于jquery.easyui.min.js实现，主要扩展功能体现在：

  • 修复了 easyui-dialog 组件的 options 方法返回的 width 和 height 属性不正确的 BUG
  • 扩展了 easyui-dialog 控件的自定义方法contentPanel，用于获取当前 easyui-dialog 控件的内容面板 panel 对象
  • 重写了 easyui-panel 控件的自定义方法 iframe，用于获取当前 easyui-dialog 控件内容面板 panel 对象中的 iframe 对象
  • 重写 easyui-dialog 控件的 refresh 方法，用于支持 iniframe 属性

依赖项

  • jquery.jdirk.js
  • jeasyui.extensions.js
  • jeasyui.extensions.menu.js
  • jeasyui.extensions.panel.js
  • jeasyui.extensions.window.js

扩展的事件

该扩展增加定义了如下事件：

Name Parameters Description Returns
onSave 无 点击保存按钮触发的事件，如果该事件范围 false，则点击保存后窗口不关闭，该触发事件仅在方法 jQuery.easyui.showDialog 中被调用 null
onApply 无 关闭窗口时应触发的事件，easyui-dialog本身就有，该触发事件仅在方法 jQuery.easyui.showDialog 中被调用 null
onClose 无 点击应用按钮触发的事件，该触发事件仅在方法 jQuery.easyui.showDialog 中被调用 null
