jQuery Eayui Panel 功能扩展

该部分扩展由文件 jeasyui.extensions.panel.js 实现。

扩展概述

扩展基于jquery.easyui.min.js实现，主要扩展功能体现在：

  • 增加了 easyui-panel、easyui-window、easyui-dialog 组件在卸载时回收内存的处理，主要用于 layout、panel(及其继承组件) 使用 iframe 嵌入网页时的内存泄漏问题
  • 增加了 easyui-panel 控件的 minWidth、maxWidth、minHeight、maxHeight 扩展属性，用来控制 easyui-panel 控件的最大/小的高度/宽度
  • 重写了 easyui-panel 控件的 onResize 事件，用于扩展四个新增属性 minWidth、maxWidth、minHeight、maxHeight 的功能
  • 增加了 easyui-panel 控件的 iniframe 扩展属性，该属性表示 href 加载的远程页面是否装载在一个 iframe 中
  • 重写了 easyui-panel 控件的 refresh 方法，用于支持新增的 iniframe 属性
  • 增加了 easyui-panel 控件的 inLayout、inTabs、inAccordion、isWindow、isDialog 扩展方法，用来判断当前 easyui-panel 出于哪个 easyui 组件中
  • 增加了 easyui-panel 控件的 iframe 扩展方法，用于获取当 iniframe: true 时当前 panel 控件中的 iframe 容器对象
  • 重写了 easyui-panel 控件的 header 方法，支持位于 easyui-tabs 中的 tab-panel 部件获取 header 对象
  • 重写了 easyui-panel 控件的 setTitle 方法，支持位于 easyui-tabs 中的 tab-panel 部件设置 title 操作
