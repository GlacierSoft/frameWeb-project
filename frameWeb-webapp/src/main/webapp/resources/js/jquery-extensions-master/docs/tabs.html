jQuery Eayui Tabs 功能扩展

该部分扩展由文件 jeasyui.extensions.tabs.js 实现。

扩展概述

扩展基于jquery.easyui.min.js实现，主要扩展功能体现在：

  • 增加了 easyui-tabs 控件的lineHeight扩展属性，用于控制当前选项卡标题栏和选项卡的 pane-body 之间的空白区域高(宽)度(px)
  • 增加了 easyui-tabs 控件的扩展属性enableContextMenu、enableNewTabMenu，用于判断是否启用点击选项卡投的右键菜单以及是否启用“创建新选项卡”的右键菜单
  • 增加了 easyui-tabs 控件的contextMenu扩展属性，用于右键点击选项卡标题时候弹出的自定义右键菜单项内容
  • 增加了 easyui-tabs 控件的showOption扩展属性，用于当右键选项卡头时，是否显示“显示该选项卡的option”菜单项
  • 增加了 easyui-tabs 控件的onRefresh扩展事件，用于当调用 easyui-tabs 的 refresh 方法后，将触发该事件
  • 增加了 easyui-tabs 控件的onContextMenu扩展事件，用于覆盖easyui-tabs的原生事件属性onContextMenu，以支持相应扩展功能
