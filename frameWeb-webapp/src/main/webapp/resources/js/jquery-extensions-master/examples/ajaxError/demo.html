jEasyUI Generic Ajax Error Extensions

该部分扩展由文件 jeasyui.extensions.js 实现。

该扩展通过快捷便利的方式允许用户自定义 ajax 请求异常事件，该事件函数会被应用到如下范围：

  • jQuery 全局 AJAX 操作
  • easyui-form
  • easyui-combobox
  • easyui-combotree
  • easyui-combogrid
  • easyui-datagrid
  • easyui-propertygrid
  • easyui-propertygrid
  • easyui-treegrid
此处仅演示 jQuery 全局异常 和 easyui-combobox 异常。



点击该按钮后，之所以不会影响到当前界面上 easyui-combobox 的 onLoadError，是因为 $.easyui.ajaxError 改变的是 easyui 组件的默认值； 而当前的 combobox 组件在初始化时已经加载过默认值了。
